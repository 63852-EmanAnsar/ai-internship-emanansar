# Task 07 – Machine Learning Foundations

## PKCERT AI & Software Development Internship

### Objective

The objective of this task is to understand the fundamentals of machine learning, including train/test splitting, overfitting, underfitting, the bias-variance tradeoff, and Linear Regression using scikit-learn.



## Topics Covered

- Train/Test Split
- Random State and Reproducibility
- Validation Set
- Overfitting and Underfitting
- Bias-Variance Tradeoff
- Linear Regression
- Model Evaluation (MSE and R² Score)
- Cross Validation
- Learning Curve
- Polynomial Regression (Bonus)



## Libraries Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
